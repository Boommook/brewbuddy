# Hooks

Custom React hooks for Brewing Buddy.

Examples to add later:

- `useBatches()` – fetch and cache list of active batches
- `useBatch(id)` – fetch a single batch and its checks
- `useCreateBatch()` – mutation for adding a batch
- `useRecordCheck(batchId)` – mutation for recording a check

Use React Query (TanStack Query) or SWR for server state; keep forms and UI state in component state or a small store.
